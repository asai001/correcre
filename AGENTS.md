# AGENTS.md

このリポジトリでは `.ai/` 配下をエージェント間の共有コンテキストとして扱う。

## 作業開始時

以下の順で必ず確認すること。

1. `.ai/agent-rules.md` — 共通ルール
2. `.ai/handoff.md` — 前回の引継ぎ
3. 必要に応じて `.ai/` 配下の他ファイル（`current-status.md`, `next-actions.md`, `decisions.md`, `architecture.md`, `project-overview.md`）
4. 大きい論点や個別タスクがある場合は `.ai/tasks/` 配下も確認すること

## 作業終了時

`.ai/current-status.md`, `.ai/next-actions.md`, `.ai/handoff.md` を更新すること。
